## Features

### Citation Formats
- **APA 7th Edition**, **ACM**, and **ASCE** citation styles
- **BibTeX export** — ready for LaTeX / Overleaf

### Auto-fill from Identifier ✨
- Paste a **DOI** (e.g. `https://doi.org/10.1000/xyz`) → fetches full metadata from CrossRef
- Paste a **URL** → scrapes title, site name, description, and publish date
- Paste an **ISBN** → fetches book info from Open Library
- Click **Lookup** or paste and it auto-triggers

### Citation Form
- **Dynamic author form** — Add multiple contributors with Surname, First Name, and M.I.
- **Source types** — Journal, Book, Website, Conference Paper
- **Abstract / Notes field** — Attach notes or paste the paper abstract
- **PDF attachment** — Link a local PDF file (desktop app only)

### Organization & Export
- **Folder organization** — Group citations by project or subject
- **Alphabetical sorting** — Citations sorted A–Z with quick letter navigation
- **Search** — Filter citations by text or notes
- **Export** — PDF, Word (.docx or .doc), or .bib file

### 🧈 Butter — AI Citation Assistant
- Ask Butter about APA, ACM, ASCE formats and rules
- Get auto-fill tips, BibTeX/Overleaf help, and citation checking
- Typing indicator and friendly personality
- Available in both the desktop app and web app

---

## How to Use

1. Select **Source Type** and **Citation Format**
2. Paste a **DOI / URL / ISBN** and click **Lookup** to auto-fill, OR fill in fields manually
3. Click **Generate Citation**
4. Click **Add to List** to save it
5. Export your full reference list as PDF, Word, or BibTeX
